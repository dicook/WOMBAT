Interactive web graphics made easy

Carson Sievert

The web is the most popular medium for sharing interactive data visualizations thanks to the portability of the web browser and the accessibility of the internet. Unfortunately, creating interactive web graphics often requires a working knowledge of a number of complicated web technologies. In recent years, we've seen a number of projects to alleviate this burden on users, particularly for statisticians and people who analyze data. I will present some examples and lessons learned from my work on the following R packages for creating interactive web graphics: plotly, LDAvis, and animint.
